Live Build Session

Follow along. By the end you'll have a real app live on the internet.

Come with: your Claude Code subscription. That's it, we'll do the rest together.
⚡ Important: Claude will ask permission a lot. Just click Approve / Yes to everything it asks. Don't overthink it.

1 Install VS Code

code.visualstudio.com
🍎 Mac Download Universal, open it, drag Visual Studio Code into your Applications folder, then open it. If you see "are you sure you want to open it," click Open.
🪟 Windows Download the 64-bit User Installer, run it, and open VS Code. If you see "Windows protected your PC," click More info → Run anyway.

2 Install the Claude Code extension

Click the Extensions icon on the left sidebar, search Claude Code, install the one by Anthropic (blue check), then sign in with your Claude account.

Same on Mac and Windows.

3 Create a free Vercel account

vercel.com

Sign up with email or Google (not GitHub). Nothing to download.

Same on Mac and Windows.

4 Open a fresh folder

File → Open Folder, make a new empty folder on your Desktop (like my-app), and open it.

Same on Mac and Windows.

5 Open Claude Code

Click the Claude icon in the left sidebar (you don't need a file open), or open the Command Palette (Cmd+Shift+P on Mac / Ctrl+Shift+P on Windows) and type "Claude Code."

6 Build it

Paste this into Claude and swap in your idea:

Prompt 1 — Build
Build me a simple [your idea] as an index.html web app.

7 Put it live

Paste this into Claude. Then just keep approving everything it asks, and log into Vercel in the browser when it opens.

Prompt 2 — Deploy
Deploy this to Vercel. If I don't have Node or the Vercel CLI installed, install whatever is needed first. Make all reasonable choices yourself, don't stop to ask me what to do, and when you're finished, give me the live URL.
🎉 Open your live URL. That's a real app on the internet, built by asking. Want to do this yourself, step by step? That's exactly what we do inside the Leveraging AI community.
